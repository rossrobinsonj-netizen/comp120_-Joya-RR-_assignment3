# Task Manager App

## Description
This project is a simple task management application that allows users to add tasks and view them in a list.  
It is created as part of Lab Assignment 3 for COMP120 – Software Engineering Fundamentals.

---

## Features
- Add a new task
- View a list of tasks
- Delete existing tasks
  
---

## Requirements
- A modern web browser (Chrome, Edge, Firefox)

---

## How to Run the Project
1. Download or clone the project repository.
2. Open the project folder.
3. Open the `index.html` file in your web browser.

---

## How to Use the Application
1. Enter a task in the input field.
2. Click the **Add Task** button.
3. The task will appear immediately in the task list.
4. Repeat the process to add more tasks.

### Delete a Task
1. Locate the task you want to remove.
2. Click the **Delete** button next to that task.
3. The task will be removed immediately from the list.
